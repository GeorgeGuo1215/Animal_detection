================================================================================
              技术栈标注（论文参考）
================================================================================

  嵌入模型:     intfloat/multilingual-e5-small (384维, 支持中英文)
  重排模型:     BAAI/bge-reranker-large (交叉编码器)
  向量存储:     Numpy 矩阵 + 归一化点积 (cosine similarity)
  稀疏检索:     BM25 (自实现, 无外部依赖)
  融合策略:     RRF (Reciprocal Rank Fusion)
  查询改写:     模板改写 / LLM 改写 (OpenAI 兼容接口)
  分块策略:     句子边界分块 + overlap
  Agent框架:    Plan-and-Solve (规划-执行-总结)
  API框架:      FastAPI + Uvicorn
  LLM接口:      OpenAI 兼容 (DeepSeek-Chat 等)
  外部工具:     MCP (Model Context Protocol)
