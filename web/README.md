# 毫米波雷达数据处理系统 - Web版

这是一个基于Web的毫米波雷达数据处理系统，提供与Python版本相同的功能，但无需安装任何软件，直接在浏览器中运行。

## 🌟 特性

- **📁 文件上传**: 支持拖拽上传和批量处理多个.txt数据文件
- **🔄 实时处理**: 在浏览器中实时处理雷达数据
- **📊 数据可视化**: 交互式图表显示处理结果
- **📈 统计分析**: 自动计算平均值、分布等统计信息
- **💾 结果导出**: 支持导出CSV数据和PNG图表
- **⚙️ 参数调节**: 可调节算法参数以适应不同场景
- **📱 响应式设计**: 支持桌面和移动设备

## 🚀 快速开始

### 1. 打开网页
直接用浏览器打开 `index.html` 文件，或部署到Web服务器。

### 2. 上传数据文件
- 点击"选择文件"按钮或直接拖拽文件到上传区域
- 支持同时选择多个.txt格式的雷达数据文件
- 文件格式：每行包含时间戳、I通道数据、Q通道数据

### 3. 开始处理
- 点击"开始处理"按钮
- 系统会自动处理所有选择的文件
- 实时显示处理进度和状态

### 4. 查看结果
- 处理完成后自动显示结果统计
- 查看各种图表：I/Q信号、星座图、呼吸波形、心跳波形等
- 查看详细的处理结果表格

### 5. 导出结果
- 点击"导出CSV"保存处理结果数据
- 点击"导出图表"保存所有图表为PNG文件

## 📊 功能说明

### 核心算法
- **圆拟合校正**: 自动校正I/Q信号的直流偏移和幅度不平衡
- **ARCSIN相位解调**: 提取目标的微小运动信息
- **数字滤波**: 分离呼吸和心跳信号
- **FFT频谱分析**: 提取心率和呼吸频率

### 数据可视化
1. **原始I/Q信号图**: 显示原始雷达数据
2. **I/Q星座图**: 显示信号分布和圆拟合结果
3. **呼吸波形图**: 提取的呼吸信号波形
4. **心跳波形图**: 提取的心跳信号波形
5. **心率分布图**: 所有文件的心率统计
6. **呼吸频率分布图**: 所有文件的呼吸频率统计

### 统计信息
- 平均心率 (bpm)
- 平均呼吸频率 (bpm)
- 处理文件数量
- 总数据点数

## ⚙️ 参数设置

点击右侧的设置按钮可以调节以下参数：

- **采样率**: 数据采样频率 (默认100Hz)
- **心率范围**: 有效心率检测范围 (默认48-180 bpm)
- **呼吸频率范围**: 有效呼吸频率检测范围 (默认6-30 bpm)
- **滤波窗口大小**: 滑动平均滤波器窗口大小 (默认10)

## 🔧 技术实现

### 前端技术
- **HTML5**: 现代Web标准
- **CSS3**: 响应式设计和动画效果
- **JavaScript ES6+**: 现代JavaScript特性
- **Chart.js**: 图表可视化库

### 核心算法
- **圆拟合算法**: 基于最小二乘法的圆拟合
- **ARCSIN解调**: 相位差分解调算法
- **数字滤波**: 滑动平均和FIR滤波器
- **FFT变换**: Cooley-Tukey快速傅里叶变换

## 🚨 注意事项

1. **文件大小**: 建议单个文件不超过10MB，以确保良好的处理性能
2. **数据质量**: 确保数据文件格式正确，包含足够的数据点
3. **浏览器性能**: 大量数据处理可能需要较长时间，请耐心等待
4. **网络连接**: 首次加载需要下载Chart.js库，请确保网络连接正常

## 🔍 故障排除

### 常见问题

1. **文件上传失败**
   - 检查文件格式是否为.txt
   - 确认文件内容格式正确
   - 尝试减小文件大小

2. **处理结果异常**
   - 检查数据质量和完整性
   - 调整算法参数设置
   - 确认数据采样率设置正确

3. **图表显示异常**
   - 刷新页面重试
   - 检查浏览器控制台错误信息
   - 确认浏览器版本支持

4. **性能问题**
   - 减少同时处理的文件数量
   - 关闭其他浏览器标签页
   - 使用性能更好的设备

### 调试模式

打开浏览器开发者工具 (F12) 查看详细的处理日志和错误信息。
