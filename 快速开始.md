# 🚀 快速开始指南

## 最快5分钟部署（Windows）

### 步骤1：安装Python（如已安装可跳过）
1. 访问 https://www.python.org/downloads/
2. 下载并安装（记得勾选 "Add Python to PATH"）

### 步骤2：获取代码
**方式A - Git用户**：
```bash
git clone https://github.com/GeorgeGuo1215/Animal_detection.git
cd Animal_detection/web
```

**方式B - 普通用户**：
1. 访问 https://github.com/GeorgeGuo1215/Animal_detection
2. 点击 "Code" → "Download ZIP"
3. 解压并进入 `web` 文件夹

### 步骤3：启动服务器
在 `web` 文件夹打开命令提示符（CMD），运行：
```bash
python -m http.server 8000
```

### 步骤4：打开浏览器
访问：http://localhost:8000

**完成！** 🎉

---

## 💡 提示

- **推荐浏览器**：Chrome 或 Edge
- **蓝牙功能**：仅在localhost下可用
- **停止服务器**：按 `Ctrl + C`

---

## 📝 一键启动脚本（可选）

创建 `启动服务器.bat` 文件，内容：
```batch
@echo off
cd /d "%~dp0"
echo 正在启动Web服务器...
python -m http.server 8000
pause
```

双击运行即可启动！
